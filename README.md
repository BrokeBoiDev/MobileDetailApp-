
# MobileDetailApp (SwiftUI • Swift Playgrounds)
A Demo For A Mobile Detailing App Created by BrokeBoi @GhostByteLab

A simple mobile detailing booking app built with **SwiftUI** in **Swift Playgrounds on iPad**.

## What it does
- **Home tab:** business name, quick Call/Text buttons, “Why us”, and popular packages
- **Services tab:** full list of service packages with pricing and inclusions
- **Book tab:** booking form with validation + review sheet
- **Confirmation:** generates a pre-filled **SMS booking request**
